# dFNC
Here is a collection of various notebooks and scripts I use for my research in dynamic functional network connectivity (dFNC) at the Neil Harris Lab at UCLA.
Everything is still very much a work in progress!
 
Note that this repository mainly serves as a personal backup for the code I work with, which is why it is a bit unorganized at the moment.
In the future, I plan to move my code to a shared repository with other lab members working on related topics.
